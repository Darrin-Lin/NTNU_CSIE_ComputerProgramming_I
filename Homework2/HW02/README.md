先用數學推出每一個的算法使得複雜度 = O(n)，公式如下：第 $n$ 個數 $c_n\over m_n$ 是 ，第 $n$ 是 $2c_{n-1} + c_{n-2}\over 2m_{n-1} + m_{n-2}$，所以複雜度為 $O(n)$，就不用一個一個算變成 $O(n!)$。
